## Project Description

Aplicación web moderna con Quasar Framework, SSR, y API REST generada con Tsoa. Los controladores Tsoa están en src-ssr para la lógica del servidor, mientras que src contiene solo el frontend Vue.js. Incluye configuración de base de datos MySQL con TypeORM y contenedor Docker.
